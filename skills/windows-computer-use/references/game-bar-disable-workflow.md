# Game Bar disable workflow

Use this when the user asks to turn off Xbox Game Bar in Windows Settings.

## Verified path
1. Open **Settings**.
2. Go to **Gaming > Game Bar**.
3. Turn **Off** the top toggle labeled **Allow your controller to open Game Bar**.

## Verification
- Re-capture the screen after the click and confirm the toggle pill is gray and the label reads **Off**.
- Do not rely only on the active window title; confirm the state visually from the screenshot.
- If the click lands in the page but the visible state does not change, re-locate the toggle and click again instead of assuming the page was updated.
