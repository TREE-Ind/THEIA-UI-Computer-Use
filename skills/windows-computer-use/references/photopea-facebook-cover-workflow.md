# Photopea: Facebook cover workflow

Session-derived notes for building an FB cover in Photopea with UI-only control.

## Verified preset
- In **File > New**, the **Social** tab contains an **FB Page Cover** preset.
- The preset size shown in the dialog is **1640 × 664 px**.
- Selecting it populates the right-side fields with Width **1640** and Height **664**.
- The **Create** button is the final step to open the blank document.

## Practical notes
- The New Project dialog is the reliable entry point for cover work; it avoids manual canvas-size typing.
- For UI-only workflows, use the preset tile and then proceed with layered editing in the blank document.
- If a click seems to do nothing, re-capture the screen and verify whether the dialog focus changed before retrying.
